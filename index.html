LLM Fine-Tunning(RAG 챗봇, QLoRA)

 RAG 챗봇  QLoRA 메인페이지 
기술 스택 : Python, Pytorch, Llama3, Streamlit, Transformers, peft, datasets, trl
프로젝트:LLM Fine-Tunning(RAG 챗봇, QLoRA)
프로젝트 참여 인원: 개인(단독 수행)
1. RAG 챗봇
1-1. RAG 파일 업로드 및 처리
- CSV, PDF 또는 TXT 파일을 업로드하면, 파일은 임시 파일로 저장된 후 Loader를 사용하여 문서 로드
- 로드된 문서는 RecursiveCharacterTextSplitter를 통해 텍스트 분할
- 분할된 텍스트는 SentenceTransformer 모델을 사용하여 임베딩
- FAISS를 이용해 VectorStore 생성
1-2. 검색 및 답변 생성
- 자용자의 질문에 대해 VectorStore를 활용하여 관련 정보를 검색
- 검색된 정보는 LLaMA-3-Bllossom 모델을 사용하여 답변을 생성하는 데 활용
1-3. 웹 인터페이스 개발
- Streamlit 프레임워크를 사용하여 웹 기반 사용자 인터페이스 구현
1-4. 대화 저장 및 출력
- 사용자의 질문과 챗봇의 응답은 화면에 표시되며, 모든 대화는 세션 상태에 저장
2. QLoRA
2-1. 데이터 전처리
- 로드된 데이터셋의 instruction과 output 필드 결합
- 모델 학습에 필요한 텍스트 형식 생성
- 각 샘플에 EOS토큰을 추가하여 문장의 끝 명시
2-2. 대화 저장 및 출력
- 모델을 4bit 양자화하여 메모리 사용량 최적화
- LoraConfig를 설정하여 모델 학습 시 미세 조정
- 학습 파라미터는 TrainingArguments를 통해 세밀하게 조정
- 이를 기반으로 SFTTrainer를 사용하여 모델을 효과적으로 학습
2-3. 모델 병합
- Fine-tunning한 모델을 기본 모델과 병합하여 최종 모델을 생성
2-4. 모델 저장 및 배포
- 저장된 모델을 허깅페이스에 모델 업로드
